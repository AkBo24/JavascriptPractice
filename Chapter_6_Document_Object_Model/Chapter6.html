Chapter 6: Document Object Model (DOM)

What is the DOM

  • an api created by the browser that parsing through HTML code and has available events for JavaScript
  • Code found in DevTools (f12) is a visual representation of the DOM
    • Everything created in HTML becomes a "node" in the DOM
    • HTML is considered the root node from which everything branches (think a tree data structure)
    • Children of the root is the head and body
      • inside the head and body are texts like p tags & headers, called Text Node(s)
    • fixes some mistakes (syntactical errors etc) found in html
  • More importantly:
    • JavaScript manipulates the DOM
    • can insert JavaScript code into the HTML
    • Provides an API for JavaScript to use and create interactable websites (ex: the mouseenter function)
